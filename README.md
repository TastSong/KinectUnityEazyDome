KinectUnityEazyDome
===================================

总述
--------------------------------------

* 此程序实现了Kinect2与unity的简单结合
* 主要以官方的DOME为演示

实现功能
-------------------------------------

* 玩家与模型人物的动作同步
* 球跟随用户的右手移动
* 挥手切换图片
* 用手来拖拽场景中的物体

程序实现步骤
------------------------------------

* 安装[Kinect2驱动](https://www.microsoft.com/en-us/download/details.aspx?id=44561)
